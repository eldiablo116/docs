## Welcome to ARCEUS

ARCEUS is a media sharing server that provides unified access to Plex, Jellyfin, and Live TV. This documentation will help you set up, configure, and use all the features ARCEUS has to offer.

## Navigating the Documentation

All services and guides are organized in the left-hand navigation menu. You'll find:

- **Welcome**: This section contains introductory information and getting started guides
- **Plex**: Complete documentation for Plex integration, configuration, and usage
- **Jellyfin**: Guides for setting up and using Jellyfin with ARCEUS
- **TV**: Live TV setup, configuration, and streaming instructions

Use the left menu to navigate between different sections. Each service has its own dedicated section with detailed guides, configuration options, and troubleshooting information.

<Info>
The navigation menu is always visible on the left side of the page, making it easy to jump between different sections of the documentation.
</Info>
